# Content Contract

## Visuals

- Prefer 3:4 or 4:5 vertical images.
- Keep one visual system across an album while using a distinct source image for every card.
- Avoid logos, watermarks, platform UI, private faces, addresses, license plates, and embedded credentials.
- Use only assets the user owns or has permission to use.

## Copy

- Use one primary objective: save, share, comment, follow, experience, inquiry, or sale.
- Keep titles concise and traceable to the supplied scene or evidence.
- Keep the caption coherent with the images and intended audience.
- Supply five to ten relevant topic candidates; do not add unrelated trending topics.
- Use short quotations only when attribution has been verified and the use is legally appropriate.
- Do not fabricate experience, results, analytics, testimonials, sources, or platform performance.

## Handoff

- Keep title, caption, topics, comment starter, and image order in separate copy blocks.
- Use relative paths in reusable specifications and manifests.
- The user manually selects topics, collection, visibility, and publishing time on the platform.
